Uploading your code

In order to grade your assignments, Jarvis requires the following header at the top (first 5 lines) of your cpp file.
/************************
* A#: A09999999
* Course: CS1400
* CS1400 Section: 1
* CS1405 Section: 8 * HW#: 1
***********************/

Understanding the results

Style

Jarvis will eventually check style.

Compilation

Jarvis uses g++ 5.4.0. A good reference for understanding g++ compiler errors can be found here.

Output

Jarvis provides the actual and expected output for your assignment. A diff between the two is also provided.

Example: Hellohello, World!world.

The red-highlighted struck-out text should be removed, and the green-highlighted text should be added.
